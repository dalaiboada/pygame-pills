# 🔄 Conversión de Tipos (Casting)

A menudo necesitamos cambiar el tipo de un dato, especialmente cuando usamos `input()`.

| Función | Propósito | Ejemplo |
| --- | --- | --- |
| `int(x)` | Convierte `x` a un número entero. | `int("10")` → `10` |
| `float(x)` | Convierte `x` a un número decimal. | `float(5)` → `5.0` |
| `str(x)` | Convierte `x` a una cadena de texto. | `str(3.14)` → `"3.14"` |
| `list(x)` | Convierte `x` a una lista. | `list((1, 2))` → `[1, 2]` |
| `bool(x)` | Convierte `x` a booleano (`True`/`False`). | `bool(1)` → `True` |
