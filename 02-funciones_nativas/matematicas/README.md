# 🧮 Funciones Matemáticas

Python incluye funciones nativas para realizar cálculos rápidos sin necesidad de importar librerías externas.

| Función | ¿Qué hace? | Ejemplo |
| --- | --- | --- |
| **`max(x, y)`** | Devuelve el valor más grande. | `max(10, 5)` → `10` |
| **`min(x, y)`** | Devuelve el valor más pequeño. | `min(10, 5)` → `5` |
| **`abs(x)`** | Devuelve el valor absoluto (sin signo). | `abs(-7)` → `7` |
| **`sum(lista)`** | Suma todos los elementos de una lista. | `sum([1, 2, 3])` → `6` |
| **`round(x)`** | Redondea un número decimal. | `round(3.6)` → `4` |
| **`pow(x, y)`** | Eleva `x` a la potencia `y`. | `pow(2, 3)` → `8` |
