# 💡 Plus: Contadores y Acumuladores

Aunque no son operadores, utilizan el operador `+=` (aumentar) o `-=` (disminuir) para llevar el control de valores en los ciclos.

| Patrón | Lógica | Ejemplo |
| --- | --- | --- |
| **Contador** | Suma una cantidad fija (usualmente 1). | `contador += 1` |
| **Acumulador** | Suma una cantidad variable (un total). | `total += valor` |
