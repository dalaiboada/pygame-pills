# 📁 Carpeta: `03-operadores/`

## 💊 Píldora 03: Operadores en Python

Los operadores son símbolos especiales que le dicen a Python que realice operaciones matemáticas, comparaciones o combinaciones lógicas sobre los datos.

---

### 🧮 Operadores Aritméticos
Permiten realizar cálculos sobre valores numéricos.

| Operador | Operación | Ejemplo | Resultado |
| :--- | :--- | :--- | :--- |
| `+` | Suma | `5 + 2` | `7` |
| `-` | Resta | `5 - 2` | `3` |
| `*` | Multiplicación | `5 * 2` | `10` |
| `/` | División (flotante) | `5 / 2` | `2.5` |
| `//` | División entera | `5 // 2` | `2` |
| `%` | Módulo (residuo) | `5 % 2` | `1` |
| `**` | Exponenciación | `5 ** 2` | `25` |

---

### ⚖️ Operadores de Comparación
Comparan dos valores y devuelven siempre un resultado booleano (`True` o `False`).

| Operador | Significado | Ejemplo | Resultado |
| :--- | :--- | :--- | :--- |
| `==` | Igual a | `5 == 5` | `True` |
| `!=` | Diferente de | `5 != 3` | `True` |
| `>` | Mayor que | `5 > 3` | `True` |
| `<` | Menor que | `5 < 3` | `False` |
| `>=` | Mayor o igual que | `5 >= 5` | `True` |
| `<=` | Menor o igual que | `5 <= 3` | `False` |

---

### 🧠 Operadores Lógicos
Se utilizan para combinar o modificar expresiones booleanas.

| Operador | Significado | Comportamiento |
| :--- | :--- | :--- |
| `and` | Y lógico | Devuelve `True` solo si **ambas** son verdaderas. |
| `or` | O lógico | Devuelve `True` si **al menos una** es verdadera. |
| `not` | Negación | Invierte el valor (`True` a `False` y viceversa). |
